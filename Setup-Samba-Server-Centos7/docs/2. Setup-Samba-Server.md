## 2.3 Setup Samba Server
- Kiểm tra gói samba hiện có nếu có sử dụng các lệnh sau.
```
rpm -qa | grep samba
yum list installed | grep samba
```
- Nếu samba được cài đặt, loại bỏ nó bằng lệnh dưới đây:
```
yum remove samba*
```
- Bây giờ, cài đặt samba bằng lệnh sau.
```
yum install samba* -y
```
